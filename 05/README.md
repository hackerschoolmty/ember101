# HSNEWS Web Client

Share links and collaborate with your Hacker School mates.

## Backend

We'll user our own installation of [HSNEWS](https://github.com/hackerschoolmty/hsnews) 
Rails app.

      $ cd hsnews
      $ foreman start

## Features

* List links
* Create links
* Update links
* Delete links
* Authentication
* Sign up
* Comment on links
* Vote on links
* See link's comments
